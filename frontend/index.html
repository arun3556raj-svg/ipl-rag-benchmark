Retrieval Pipeline


        

          Text2SQL
          

            
Question → DeepSeek → SQL → SQLite → Answer

            
Exact aggregations, rankings, counts. Deterministic once SQL is correct.

          

        


        

          Hybrid RAG
          

            
Question → BM25 + ChromaDB → RRF → 12 chunks → Answer

            
4,629 chunks. Sparse and dense retrieval fused via Reciprocal Rank Fusion.

          

        


        

          LightRAG
          

            
Question → Entity extraction → Personalised PageRank → Subgraph → Answer

            
899 nodes, 2,443 edges. Players, teams, venues and their relationships.

          

        


        

          Classifier
          

            
Question → all-MiniLM-L6-v2 → Logistic Regression → Best arch

            
90% LOO-CV accuracy on 50 labelled questions. 384-dim embeddings.
